# antoniolg’s Homebrew Tap

Homebrew tap for shipping my CLI tools.

## Install

```bash
brew tap antoniolg/tap
```

## Install Packages

```bash
# formula
brew install antoniolg/tap/<name>

# cask
brew install --cask antoniolg/tap/<name>
```

## Packages

### Formulae

- `justdoit` — Time‑blocking CLI + TUI for Google Tasks and Calendar

## Update / Uninstall

```bash
brew update
brew upgrade

brew uninstall <formula>
brew uninstall --cask <cask>
```

## Notes

- Run `brew info antoniolg/tap/<name>` for per‑tool caveats (permissions, setup steps, etc.).
